Slider

Owl Carousel is most modern touch enabled jQuery plugin that lets you create a beautiful responsive carousel slider. Creating a slider with owl carousel is really simple. Just copy a slider demo that suits your need. copy and paste the setup jquery code in main.js file. Thats it.
