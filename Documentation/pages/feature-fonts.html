Fonts

To set your custom google font go to Google Fonts then pick your favorite font(s) and then copy the embed link and finally replace with the already set google font.


Go to style.css (line no 55) and change it to your font-family

@import url('https://fonts.googleapis.com/css?family=Roboto:100,300,400,400i,500,700,900&display=swap');

Working with CSS

Then, go to style.css file and change font-family value font-family: 'Roboto', sans-serif; 

body {
    font-family: 'Roboto', sans-serif;
    font-weight: normal;
    font-style: normal;
    font-size: 14px;
    -webkit-font-smoothing: antialiased;
    -moz-osx-font-smoothing: grayscale;
    letter-spacing: .4px
}

h1,
h2,
h3,
h4,
h5,
h6 {
    font-family: 'Roboto', sans-serif;
    margin-top: 0px;
    margin-bottom: 0px;
    line-height: 26px;
}
